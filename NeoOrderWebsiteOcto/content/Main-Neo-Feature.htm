Cahaya dan Suara

Selain output suara, kelebihan lain dalam 8BitMixtapeNEO ini terletak pada 8 lampu RGB (Red, Green, Blue) Neo Pixels yang dapat menghasilkan cahaya warna-warni cemerlang.

Kemudahan Modifikasi

Kemudahan modifikasi alat ini juga telah dipersiapkan sejak awal dengan menyediakan ruang-ruang di dalam papan PCB sehingga pengguna ahli dapat menambah fitur sesuai keinginan. 

Banyak Variasi

Dengan adanya kemudahan untuk mengunggah kode, kalian dapat mengganti variasi fungsi suara baik dengan mengunggah kode program sendiri, atau menggunakan kode program yang telah disediakan developer 8BitMixtapeNEO.

Open Source

Yang lebih seru lagi, semua skema dan kode program yang dibutuhkan untuk merakit sendiri 8BitMixtapeNEO tersedia secara open source / bebas bagi siapa saja untuk dibuat ulang atau dimodifikasi.
